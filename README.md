# Tavolo Marketing Site

Marketing website and landing pages for Tavolo.

## Pages

- `/` - Homepage
- `/features` - Product features
- `/pricing` - Pricing page
- `/blog` - Blog posts
- `/compare/opentable` - Tavolo vs OpenTable
- `/compare/resy` - Tavolo vs Resy
